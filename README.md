# Sincify

Sincify is a collaborative online whiteboard where people can anonymously draw, sketch ideas, and brainstorm together in real time.

## Why I Built This

I built Sincify to make idea sharing fast and simple without login friction.  
The goal is to let anyone open a board, share a link, and start collaborating instantly with privacy-focused real-time drawing.

## What It Does

- Real-time multi-user whiteboard collaboration
- Public boards, private encrypted boards, and named boards
- Freehand drawing and shape tools
- Text tool with inline editing
- Eraser and clear-canvas actions
- Export canvas to image
- Responsive UI for desktop and mobile
- Light and dark theme support
- End-to-end encrypted room key flow via URL hash fragment
- Auto-reconnect friendly message queue behavior
